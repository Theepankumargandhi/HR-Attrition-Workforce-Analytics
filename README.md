
# HR Attrition and Workforce Analytics Dashboard

## Overview
This project analyzes employee data to understand workforce trends, attrition patterns, and demographic insights. 
The objective was to explore factors affecting employee turnover and present findings through interactive dashboards.

## Problem Statement
Organizations need better visibility into employee attrition and workforce distribution. 
This project focuses on analyzing employee records to uncover patterns in attrition and key workforce metrics.

## Tools Used
- SQL
- Power BI
- Tableau
- Excel
- Python

## Dataset
The dataset contains employee information such as:
- Department
- Age
- Gender
- Education
- Job Role
- Attrition status

Data was cleaned and transformed before analysis.

## Project Workflow
1. Cleaned and prepared data using Excel and Python
2. Performed SQL analysis to calculate attrition rates and trends
3. Built dashboards in Power BI and Tableau
4. Generated insights to support decision making

## Key Insights
- Identified overall attrition rate and department-wise variations
- Analyzed workforce demographics and distribution patterns
- Highlighted departments with higher employee turnover

## Dashboard Features
- Attrition rate KPI
- Department-wise analysis
- Workforce demographic breakdown
- Interactive filters for exploration

## Conclusion
This project demonstrates practical skills in data cleaning, SQL analysis, and dashboard development, 
showing how analytics can support workforce planning and business decisions.
